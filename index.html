¡Bienvenido al Workshop de Media Queries!

¿Qué son las Media Queries?

Las Media Queries son una técnica de CSS que nos permite aplicar estilos dependiendo de ciertas características del dispositivo en el que se está visualizando la página.

¿Por qué son importantes?

Las Media Queries son importantes porque nos permiten adaptar nuestra página web a diferentes dispositivos y tamaños de pantalla. Esto es fundamental para que nuestra página se vea bien en cualquier dispositivo.

¿Cómo se usan?

Las Media Queries se usan en CSS con la regla{' '} @media. Dentro de esta regla, podemos especificar diferentes características del dispositivo en el que se está visualizando la página, como el ancho de la pantalla, la orientación del dispositivo, la densidad de píxeles, etc.

Ejemplo

Por ejemplo, si queremos que un texto sea de color rojo en pantallas con un ancho de 600px o más, podemos hacerlo de la siguiente manera: 

                    
                        {`@media (min-width: 600px) {
    p {
        color: red;
    }
}`}
